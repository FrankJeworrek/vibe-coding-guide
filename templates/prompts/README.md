# Vibe Coding Prompts

This directory contains standard prompt templates for the Agnostic Vibe Coding workflow.

## Usage
1. Open the desired template.
2. Copy the raw text.
3. Paste it into your AI chat (Gemini, Claude, ChatGPT).
4. Fill in the bracketed `[ ]` sections.

## Templates
- **continue-session.md**: Standard prompt to resume work.
- **debug-issue.md**: Optimized prompt for troubleshooting.
